## Speedrun History
<Tabs>
<TabItem label="Merciless, True Ending - PC">
        <Steps>
            1. Persona 3 Reload was released worldwide on February 2, 2024.
            2. The first recorded run was done by TatsuyaAct with a time of [**10:30:14**](https://www.youtube.com/watch?v=z3gDQlx5ih4)
            3. On 2024/03/25, TatsuyaAct improved the record to a time of [**9:35:53**](https://www.youtube.com/watch?v=GFhfFDzL_rs)
            4. On 2024/06/1, Michin11 would improve the record by close to an hour with a time of [**8:39:04**](https://www.youtube.com/watch?v=L38z_69ISPQ)
            5. On 2024/7/12, LeftBishop set a new world record with a [**8:34:21**](https://www.twitch.tv/videos/2195056311)
            6. On 2024/07/25, Darkkefka claimed the world record with a time of [**8:25:23**](https://youtu.be/cm1LeKbBolM)
            7. On 2024/08/06, LeftBishop stole back the record to a time of [**8:01:28**](https://www.twitch.tv/videos/2217466037)
            8. The current world record holder is Michin11 with a time of [**7:48:45**](https://youtu.be/w6CDMH14qHg)
        </Steps>
    </TabItem>
    <TabItem label="Merciless, True Ending - Console">
        <Steps>
           1. The first recorded run was done by Liv with a time of [**10:56:27**](https://www.youtube.com/watch?v=DaB8OjEgN5c)
	   2. 2 weeks later, Liv would improve the record to [**9:51:53**](https://www.youtube.com/watch?v=tmguH8KVS0U)
	   3. On 2024/4/6, DarkKefka set his first time of, and improved the record to a [**9:34:56**](https://www.youtube.com/watch?v=iFnujkL72ds)
	   4. The current world record holder is LillieZora with a time of [**9:18:57**](https://www.twitch.tv/videos/2287314052)
        </Steps>
    </TabItem>
<TabItem label="Heartless, Episode Aigis - PC">
        <Steps>
            1. The Episode Aigis DLC for Persona 3 Reload was released worldwide on September 10, 2024.
            2. The first recorded run was done by LeftBishop with a time of [**4:55:59**](https://youtu.be/4WKWpzH33cA)
            3. On 2021/11/05, Darkkefka set a new record with a time of [**4:50:27**](https://youtu.be/deXkSH6i4_k)
            4. On 2024/11/21, LeftBishop updated the record with a time of [**4:40:02**](https://www.twitch.tv/videos/2307483441)
            5. The current world record holder is DarkKefka with a time of [**4:35:44**](https://www.twitch.tv/videos/2369543217)
        </Steps>
    </TabItem>
