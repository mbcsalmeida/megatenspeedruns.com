## Speedrun History
<Tabs>
    <TabItem label="Any% - Easy - PC">
        <Steps>
            1. Persona 5 Strikers was released worldwide on February 22, 2021.
            2. The first recorded run was done by TatsuyaAct with a time of [**5:08:17**](https://www.twitch.tv/videos/925716221). He would improve it to a [**4:25:29**](https://www.twitch.tv/videos/990207868).
            3. On 2024/07/19, Rodillo would set a new record with a time of [**4:25:03**](https://youtu.be/hJTo_LXFQFc). He would improve the record to a [**4:16:40**](https://www.twitch.tv/videos/2217608430).
            2. The current world record holder is TatsuyaAct with a time of [**4:10:59**](https://youtu.be/hSTqwU9EVxo).
        </Steps>
    </TabItem>
    <TabItem label="Any% - Easy - Console">
        <Steps>
            1. Persona 5 Strikers was released worldwide on February 20, 2020.
            2. The first run available was done by Worldie with a time of [**7:27:38**](https://www.twitch.tv/videos/796515294). He brought the time down to [**5:07:33**](https://youtu.be/Rx5Ah09U2Hk).
            3. On 2022/07/23, Jordiee set a new record with a time of [**4:57:47**](https://youtu.be/8fHAQq8vnJY)
            4. The current world record holder is Jordiee with a time of [**4:37:52**](https://youtu.be/wDxsbOBD1H8)
        </Steps>
    </TabItem>
    <TabItem label="Any% - Hard - PC">
        <Steps>
            1. Persona 5 Strikers was released worldwide on February 22, 2021.
            2. The first recorded run was done by TatsuyaAct with a time of [**9:39:07**](https://www.twitch.tv/videos/767958327) on 2020/10/11. He would improve it to a [**7:35:36**](https://www.twitch.tv/videos/941146062) 4 months later.
            3. On 2021/03/20, HeroMystic would set a new record with a time of [**6:25:29**](https://youtu.be/BD11kGD5Jos). 
            4. The current world record holder is HeroMystic with a time of [**6:10:32**](https://youtu.be/FF_drbeWdWs).
        </Steps>
    </TabItem>
    <TabItem label="Any% - Hard - Console">
        <Steps>
            1. Persona 5 Strikers was released worldwide on February 20, 2020.
            2. The first recorded run was done by HeroMystic with a time of [**7:55:21**](https://youtu.be/EkIeQ0KWOLE).
            3. The current world record holder is TatsuyaAct with a time of [**6:53:40**](https://www.twitch.tv/videos/1098134153)
        </Steps>
    </TabItem>
</Tabs>
