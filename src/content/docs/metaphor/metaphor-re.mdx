## Speedrun History
<TabItem label="Hard, True Ending - PC">
<Steps>
1. Metaphor:ReFantazio was released worldwide on October 11th, 2024.
2. The first ever recorded run was done on 2024/11/16 by mulsqi with a time of [**7:45:29**](https://www.twitch.tv/videos/2304147076)
3. On 2024/11/25, Neviutz improved the record to [**7:00:32**](https://www.youtube.com/watch?v=UxnFYN7ZtMI)
4. On 2024/12/14, mulsqi improved the record to [**6:30:18**](https://www.twitch.tv/videos/2334634021) 
5. On 2024/12/21, ChampionBeef improved the record to [**6:16:18**](https://www.twitch.tv/videos/2335013593)
6. The current world record on PC is held by ChampionBeef with a time of [**6:00:32**](https://www.twitch.tv/videos/2342342200), set on 2025/1/1
</Steps>
</TabItem>
<TabItem label="Hard, True Ending - Console">
<Steps>
